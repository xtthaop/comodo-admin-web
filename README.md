### 简介

Comodo Admin 是一款极简的 RBAC（基于角色的访问控制） 后台管理系统，前端基于 Vue 3 开发，后端基于 PHP 8.1 开发。它提供了一种 RBAC 后台管理系统的具体实现思路和运行框架。

### 在线调试：<https://cnb.cool/tidalwhisper/comodo-open/comodo-admin-web>

点击云原生开发按钮，等待环境准备完成后点击在线 WebIDE 会打开 VSCode 的网页版, 等待右下角弹出一个有 Open 按钮的弹出框，点击即可在浏览器中打开一个完整的开发环境，无需安装任何软件。

环境准备过程参考 [./.ide/Dockerfile](./.ide/Dockerfile) 和 [.cnb.yml](.cnb.yml)。

初始账号信息与在线预览相同。
