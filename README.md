## 简介

Comodo Admin 是一款极简的 RBAC（基于角色的访问控制） 后台管理系统，前端基于 Vue 3 开发，后端基于 PHP 8.1 开发。它提供了一种 RBAC 后台管理系统的具体实现思路和运行框架。

[系统预览](https://zxctb.top:8081)  
[使用文档](./PROMPT.md)



## 快速开始

#### 安装依赖
```
pnpm install
```

#### 本地开发环境运行
```
pnpm run dev
```

#### 打包到生产环境
```
pnpm run build
```
